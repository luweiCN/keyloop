# KeyLoop 训练领域术语 (Context Glossary)

KeyLoop 是一个终端打字训练工具。本文件只定义领域语言，不含实现细节。

## 薄弱诊断 (Weakness Diagnosis)

从用户练习的字符级错误与速度，判断其薄弱的**键位、手、字符类**（数字 / 符号 / 大小写）与**技能维度**（词流畅度 / 长词 等）。

薄弱诊断的产物用于决定"**练什么**"——驱动各形态的练习配比，以及靶向训练层的选材。它**不**用于把用户打错过的具体句子或段落重新拿来反复练（见 错题回流）。

## 靶向训练层 (Targeted Layer)

针对诊断出的薄弱**键 / 字符 / 词**的练习，粒度是键、字符、单词。包括基础训练（按薄弱键加权）与单词段（薄弱词优先）。在这一层，"用薄弱诊断指导选材"是正当且核心的做法。

## 薄弱词 (Weak Word)

用户**累计多次**打错的词（窗口内反复错，而非单次偶然手滑）。属于靶向训练层的选材依据：单词段优先安排薄弱词。

与错题回流的区别：它把"词"作为薄弱**单元**来诊断与选材，而非把某次错过的整句原样重复。判定薄弱词需要"多次错"的门槛，避免把一次手滑误当薄弱。

## 综合应用层 (Applied Layer)

句子与文章练习。提供自然、连贯、多样的真实文本，用于综合运用与维持手感。

**这一层不靶向特定薄弱点**：一篇文章里错几个字符是正常的，翻篇即过；薄弱键会在自然文本中被自然练到，无需刻意编排或重复。

## 错题回流 (Mistake Replay) — 反模式

把用户曾打错的具体内容（尤其整句 / 整段）重新作为练习材料反复重练。

在综合应用层，这被明确认定为**反模式**——"打字不是做题，不搞错题集"。用户的某次错误不代表该句子是其薄弱所在；薄弱应通过薄弱诊断在键 / 字符 / 词层面体现，而非重复整句。
